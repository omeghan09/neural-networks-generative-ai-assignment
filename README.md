The baseline CNN achieved a test accuracy of 0.8989, while the model trained with data augmentation achieved 0.8831. This represents a slight decrease of 0.0158 in accuracy. While data augmentation is typically used to improve generalization, in this case it slightly reduced performance, likely because the dataset is already relatively clean and simple. The added variability from augmentation may have made learning more difficult within the limited number of training epochs. I only used 5.  Performance may have increased if I used more.
